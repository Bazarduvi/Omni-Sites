Política de Privacidad

NexusAI almacena datos localmente en tu dispositivo mediante IndexedDB y LocalStorage. No enviamos datos personales a servidores externos sin tu consentimiento. Las API keys se guardan localmente y nunca se comparten. Al usar APIs de terceros (Groq, Gemini, etc.), los datos se envían a sus servidores según sus políticas.
